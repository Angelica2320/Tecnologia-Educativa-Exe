Reflexión y Reutilización

Reflexión

Este OVA fue diseñado siguiendo el modelo ADDIE para garantizar una secuencia lógica y pedagógica del aprendizaje.

Se priorizó el aprendizaje activo mediante actividades interactivas y producción de oraciones, ya que el uso del idioma es más efectivo que la simple memorización.

La evaluación fue diseñada para medir el uso real de Do y Does en oraciones afirmativas, negativas y preguntas, lo cual está alineado con el objetivo de aprendizaje.

El uso de eXe-Learning permite crear un recurso accesible, flexible y reutilizable, fortaleciendo la integración de las TIC en el proceso educativo.

Objeto Virtual de Aprendizaje (OVA) fue diseñado de forma modular y reutilizable.

Puede adaptarse a otros contextos educativos porque:
- El contenido puede cambiarse fácilmente para otros tiempos verbales (past, future, modal verbs).
- El vocabulario puede sustituirse por temas como profesiones, rutinas, familia o comida.
- Puede usarse en diferentes niveles educativos (primaria, secundaria o educación de adultos).
- Puede integrarse en plataformas como Moodle, Google Classroom o usarse de manera autónoma.

Esto permite que el OVA no sea exclusivo de una sola clase, sino que pueda reutilizarse en múltiples cursos de inglés.
